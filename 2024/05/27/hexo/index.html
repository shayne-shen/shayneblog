hexo 終端指令筆記

#＃環境
1.node -v 確認node.js版本及是否成功安裝
2.npm -v 確認npm版本及是否成功安裝
3.sudo npm install -g hexo-cli 安裝hexo(sudo為mac必備前綴指令)

#＃HEXO指令
1.cd 前往
2.hexo init[網站名稱（英文）]
3.hexo server
4.hexo new[‘文章名稱’]
5.hexo new page[頁面名稱]
6.ctrl+c
7.hexo clean
8.hexo generate
9.hexo deploy
